<!-- Features -->

### Key Features <a name="key-features"></a>

> Description of 1-3 key features of the application.

- **When the page loads, the webapp shows the list of Tv series that could be filtered out by some parameter**
- **These Tv series are gotten from an API and when the user clicks (or taps) on a series, the application navigates to the details page.**
- **On the details page, the webapp retrieves data from the API to show detailed data of the series that was clicked on and when the user clicks on the "Back" button , the user navigates back to the home page**

### Prerequisites

In order to run this project you need:

- A Desktop or Laptop Computer running on with Windows, Mac OS or Linux operating system.
- Git installed on the Computer
- Nodejs installed on the Computer
- A browser installed on the Computer

### Setup

Clone this repository to your desired folder:

- `https://github.com/Masouma864/TvSeriesSearch` 
- create a directory in your preferred location (e.g in the Desktop or Document directory)


### Install

Install this project with:

- Dependencies _(a packet manager is required)_: then you run the command `npm install` in the terminal

### Usage

To run the project, execute the following command:

- `npm run start`
- This will run the project on localhost:3000

### Run tests

To run tests, run the following command:

- `npm run test`
- This will run all tests on your local machine
